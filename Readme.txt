坦克大战功能说明：
  程序是对经典坦克大战的抄袭＋超简易版
  一共分为三个屏幕：
  屏幕一：有两个选项1.player1；2.hello！
     点击选项1出现屏幕二，游戏界面
     点击选项2出现屏幕三，hello界面（对spritekit框架的基本探索）
  屏幕二：屏幕左边是方向控制，右边是发射子弹，中间是游戏地图
     操作很简单，点击左右的按钮即可；
     游戏说明，敌机不能发子弹，所以只要干掉四个敌机就会出现you win！字样。如果自己干掉了boss就失败了，出现gameover字样。
  屏幕三：显示了hello world字样！这个是当时为了测试spritekit如何显示来写的，然后顺手保留了这个界面
